# Django Notes App

This is a simple notes management web application built using Django.

The project was created as a practice project to understand Django fundamentals such as models, views, templates, authentication, and CRUD operations.

## Features

- User Registration and Login
- Create Notes
- View Notes
- Update Notes
- Delete Notes
- Notes are stored separately for each logged-in user
- Automatic date and time recorded when a note is created
- Simple Bootstrap-based user interface

## CRUD Functionality

The application implements full CRUD operations:

- **Create** – Users can create new notes.
- **Read** – Users can view their saved notes.
- **Update** – Existing notes can be edited.
- **Delete** – Notes can be removed.

## Technologies Used

- Python
- Django
- HTML
- Bootstrap
- SQLite

## Purpose

This project was built for learning and practicing Django web development concepts including authentication, database models, and CRUD operations.
